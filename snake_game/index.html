Rules

This is the same as the original snake game. Get to the red square using the wasd keys, arrow keys, or the buttons for those who don't have a keyboard (sorry for the horrible controls). The snake grows longer as you eat the red square. Try to not hit yourself. One thing that is different in this version of the game is that you can go through the walls. When you go through the wall, you go to the opposite side. Good Luck
