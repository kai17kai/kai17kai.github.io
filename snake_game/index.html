Rules

This is the same as the original snake game. Get to the red square using the wasd keys, arrow keys, the buttons for those who don't have a keyboard (sorry for the horrible controls), or a xbox controller. The snake grows longer as you eat the red square. Try to not hit yourself. There are two differences though between this game and the original snake game.This version of the game allows you to go through the walls and the snake speeds up slowly as you eat the red square. When you go through the wall, you go to the opposite side. Good Luck. Try to get to a score of 175 because there is a special prize.
